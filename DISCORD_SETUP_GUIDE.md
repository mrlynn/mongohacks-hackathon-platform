### Option 1: Use Your Personal Server (Easiest)

**Step 1: Create a channel for your team**
1. Open Discord
2. Go to your personal server (or any server you're in)
3. Right-click on the channel list â†’ "Create Channel"
4. Name it: `hackathon-team-[your-team-name]`
5. Make it private (toggle "Private Channel")
6. Click "Create Channel"

**Step 2: Generate invite link**
1. Right-click on the new channel â†’ "Edit Channel"
2. Go to "Permissions" tab
3. Click "Create Invite" or click the channel â†’ "Invite" button
4. Set expiration: **7 days** (or custom for hackathon duration)
5. Set max uses: **10** (or match your team size)
6. Copy the invite link: `https://discord.gg/abc123`

**Step 3: Add to your team**
1. Go to MongoHacks team page or creation form
2. Select "Discord" as communication platform

4. Save!

### Option 2: Create a Dedicated Server (More Control)

**Step 1: Create a new Discord server**
1. Open Discord
2. Click the "+" button on the left sidebar
3. Click "Create My Own"
4. Choose "For a club or community"
5. Name it: `[Your Team Name] - MongoHacks 2026`
6. Upload team logo/icon (optional)
7. Click "Create"

**Step 2: Set up channels**
Discord creates default channels. Customize them:
- `#general` â†’ Rename to `#team-chat`
- `#random` â†’ Delete or rename to `#ideas`
- Add `#code` channel for code snippets
- Add `#resources` channel for links/docs
- Voice channel: Already created ("General")

**Step 3: Generate invite link**
1. Click "Invite People" button (top-right)
2. Click "Edit invite link"
3. Set expiration: **7 days**
4. Set max uses: **10**
5. Copy the link: `https://discord.gg/xyz789`

**Step 4: Add to your team**
1. Go to MongoHacks team page
2. Select "Discord" as communication platform
3. Paste the invite link
4. Save!

## ğŸ›¡ï¸ Discord Privacy & Security

**Recommended settings for team channels:**

1. **Private Channel** (in personal server):
   - Only invited members can see/join
   - No one else in your server can access

2. **Temporary Invite Links**:
   - Set expiration (7 days for hackathon)
   - Set max uses (match team size)
   - Prevents random people joining

3. **Permissions**:
   - All team members can send messages
   - All can share files
   - All can use voice
   - Only you (leader) can delete channels
